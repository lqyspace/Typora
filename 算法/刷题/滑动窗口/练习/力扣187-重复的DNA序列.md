# 187-重复的DNA序列

[力扣187-重复的DNA序列](https://leetcode.cn/problems/repeated-dna-sequences/description/)，难度中等

![image-20230821102708118](https://raw.githubusercontent.com/lqyspace/mypic/master/PicBed/202308211027284.png)

滑动窗口：

```java
class Solution {
    public List<String> findRepeatedDnaSequences(String s) {
        int n = s.length();
        // 将字符转换为数字
        int[] nums = new int[n];
        for(int i=0; i<n; i++){
            switch(s.charAt(i)) {
                case 'A':
                    nums[i] = 0;
                    break;
                case 'C':
                    nums[i] = 1;
                    break;
                case 'G':
                    nums[i] = 2;
                    break;
                case 'T':
                    nums[i] = 3;
                    break;
            }
        }
        
        // 记录重复值
        Set<Integer> seen = new HashSet<>();
        // 记录结果
        Set<String> res = new HashSet<>();
        
        // 记录位数
        int L = 10;
        // 进制
        int R = 4;
        // 存储 R^(L-1)
        int RL = (int)Math.pow(R, L-1);
        // 记录滑动窗口内的字符的哈希值
        int windowHash = 0;
        
        int left = 0, right = 0;
        while(right < n){
            windowHash = windowHash * R + nums[right++];
            
            if(right - left == L){
                if(seen.contains(windowHash))
                    res.add(s.substring(left, right));
                else 
                    seen.add(windowHash);
                
                windowHash -= nums[left++] * RL;
            }
        }
        return new LinkedList<>(res);
    }
}
```
