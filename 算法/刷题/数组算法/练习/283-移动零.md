# 283-移动零

[力扣283-移动零](https://leetcode.cn/problems/move-zeroes/)，难度简单

![image-20230818141503211](https://raw.githubusercontent.com/lqyspace/mypic/master/PicBed/202308181415435.png)

```java
class Solution {
    public void moveZeroes(int[] nums) {
		int n = nums.length;
        int fast, slow;
        fast = slow = 0;
        while(fast < n){
            if(nums[fast] != 0)
                nums[slow++] = nums[fast++];
            else
                fast++;
        }
        while(slow<n)
            nums[slow++] = 0; 
    }
}
```
