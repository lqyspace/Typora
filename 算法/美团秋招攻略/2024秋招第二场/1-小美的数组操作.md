为了理解题解：特提供一个实例供思考：

```
6
7 9 11 3 21 5
```

代码如下：

```java
import java.util.*;

// 注意类名必须为 Main, 不要有任何 package xxx 信息
public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        int[] nums = new int[n];
        for(int i=0; i<n; i++){
            nums[i] = in.nextInt();
        }

        // 如果所有的数字加起来恰好是n的倍数，那么肯定可以归为同一个数字
        // 否则，我们是一定可以产出一个n-1个众数
        // 关键是排除哪一个数字？
        // 排除的数字要是最好的操作次数尽可能少，显然应该排除最大值或者最小值（与平均数的偏移最多）
        
        
        // 为什么是在n-1个数中产生众数是合理的：
        // 首先我们需要明白，题目是让我们求解操作次数，并不是求均值
        // 其次，我们通过求解n-1的平均数，然后计算正数差值总和x，以及负数差值总和y，x和y的差距大小至少为0，我们取x和y中的最大值作为操作次数
        // 有的同学就会反驳了，x和y的值并不代表了真正的操作次数
        // 但是，我们应该想一想，我们处理的是n-1个数，x和y的操作次数的差值完全可以施加在最后一个数上，因此取x和y中的最大值作为操作次数是合理的
        long sum = 0;
        for(int num: nums){
            sum+=num;
        }
        if(sum%n==0){
            System.out.println(dp(nums, 0, n-1, sum/n));
            return;
        }
        Arrays.sort(nums);
        // 考虑到平均数可能除不尽，所以这里用两个平均数
        long m1 = Math.min( dp(nums, 1, n-1, (sum-nums[0])/(n-1)), dp(nums, 1, n-1, (sum-nums[0])/(n-1)+1) );
        long m2 = Math.min( dp(nums, 0, n-2, (sum-nums[n-1])/(n-1)), dp(nums, 0, n-2, (sum-nums[n-1])/(n-1)+1) );

        System.out.print(Math.min(m1, m2));
    }

    public static long dp(int[] nums, int left, int right, long avg){
       long x = 0, y = 0;
        for(int i=left; i<=right; i++){
            if(nums[i]>avg){
                x += (nums[i] - avg);
            }else {
                y += (avg - nums[i]);
            }
        }
        return Math.max(x, y);
    }
}
```
