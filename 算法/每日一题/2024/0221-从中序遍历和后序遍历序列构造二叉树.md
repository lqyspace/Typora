# 106-从中序和后序遍历序列构造二叉树

[106-从中序和后序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-inorder-and-postorder-traversal/description/)，难度中等

![image-20240221111122605](https://fastly.jsdelivr.net/gh/lqyspace/mypic@master/img1/202402211111674.png)

题解：**递归**

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public TreeNode buildTree(int[] inorder, int[] postorder) {
        if(inorder==null || inorder.length==0) return null;
        TreeNode root = new TreeNode(postorder[postorder.length-1]);
        int inde = returnIndex(inorder, postorder[postorder.length-1]);
        root.left = buildTree(Arrays.copyOfRange(inorder, 0, inde), Arrays.copyOfRange(postorder, 0, inde));
        root.right = buildTree(Arrays.copyOfRange(inorder, inde+1, inorder.length), Arrays.copyOfRange(postorder, inde, postorder.length-1));
        return root;
    }

    public int returnIndex(int[] nums, int target){
        for(int i=0; i<nums.length; i++){
            if(target==nums[i])
                return i;
        }
        return -1;
    }
}
```

时间复杂度：$O(N^2)$

空间复杂度：$O(N^2)$



题解：**哈希表+递归**

用一个哈希表记录`inorder`中的值在`inorder`哪一个位置；再用递归遍历。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public TreeNode buildTree(int[] inorder, int[] postorder) {
        if(inorder==null || inorder.length==0) return null;
        int n = inorder.length;
        Map<Integer, Integer> map = new HashMap<>(n);
        for(int i=0; i<n; i++){
            map.put(inorder[i], i);
        }
        return dfs(inorder, 0, n, postorder, 0, n, map);
    }

    private TreeNode dfs(int[] inorder, int inL, int inR, int[] postorder, int postL, int postR, Map<Integer, Integer> map){
        if(inL==inR) return null;
        // if(inL+1==inR) return new TreeNode(inorder[inL]); // 叶子节点，可以省略
        int leftSize = map.get(postorder[postR-1])-inL;
        TreeNode left = dfs(inorder, inL, inL+leftSize, postorder, postL, postL+leftSize, map);
        TreeNode right = dfs(inorder, inL+leftSize+1, inR, postorder, postL+leftSize, postR-1, map);
        return new TreeNode(postorder[postR-1], left, right);
    }
}
```
