# 889-根据前序和后序遍历构造二叉树

[889-根据前序和后序遍历构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-postorder-traversal/description/)，难度中等

![image-20240222125033529](https://fastly.jsdelivr.net/gh/lqyspace/mypic@master/img1/202402221250604.png)

题解：**哈希表+递归**

- 由于这道题只提供了前序和后序遍历，所以构造的二叉树并不唯一
- 我们可以把前序数组的第二个数字作为左子树的根，因此在后序数组中我们可以确定左子树
- 

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public TreeNode constructFromPrePost(int[] preorder, int[] postorder) {
        if(preorder==null || preorder.length==0) return null;
        Map<Integer, Integer> map = new HashMap<>();
        int n = postorder.length;
        for(int i=0; i<n; i++){
            map.put(postorder[i], i);
        }
        return dfs(preorder, 0, n, postorder, 0, n, map);
    }
    private TreeNode dfs(int[] preorder, int preL, int preR, int[] postorder, int postL, int postR, Map<Integer, Integer> map){
        if(preL==preR) return null; // 空节点
        if(preL+1==preR) return new TreeNode(preorder[preL]); // 叶子节点，需要对叶子节点单独讨论
        int leftSize = map.get(preorder[preL+1])-postL+1;
        TreeNode left = dfs(preorder, preL+1, preL+1+leftSize, postorder, postL, postL+leftSize, map);
        TreeNode right = dfs(preorder, preL+1+leftSize, preR, postorder, postL+leftSize, postR-1, map);
        return new TreeNode(preorder[preL], left, right);
    }
}
```

题解：**递归**

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public TreeNode constructFromPrePost(int[] preorder, int[] postorder) {
        if(preorder==null || preorder.length==0) return null;
        if(preorder.length==1) return new TreeNode(preorder[0]); // 叶子节点
        int leftSize = indexOf(postorder, preorder[1])+1;
        TreeNode left = constructFromPrePost(Arrays.copyOfRange(preorder, 1, leftSize+1), Arrays.copyOfRange(postorder, 0, leftSize));
        TreeNode right = constructFromPrePost(Arrays.copyOfRange(preorder, leftSize+1, preorder.length), Arrays.copyOfRange(postorder, leftSize, preorder.length-1));
        return new TreeNode(preorder[0], left, right);
    }

    private int indexOf(int[] a, int x){
        for(int i=0; i<a.length; i++)
            if(a[i]==x) return i;
        return -1;
    }
}
```
